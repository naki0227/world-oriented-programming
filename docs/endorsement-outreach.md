# Endorsement Outreach

This note packages the shortest public materials to send when asking for an arXiv endorsement.

## Short English Message

```text
Dear Professor [Name],

I am preparing to submit a paper to arXiv in cs.PL (Programming Languages).
The paper presents sekai, an experimental world-oriented programming language in which programs describe executable worlds through space, time, and constraints.

The project already includes a prototype DSL and runtime, a viewer and diagram-to-logic round-trip, a first semantics trajectory, underdetermined-world convergence slices, and a first visibility-based geometry pillar.

Because this is my first submission to arXiv in cs.PL, arXiv requires an endorsement from a qualified submitter.
If you are eligible and willing, I would be very grateful if you could endorse my submission.

I can send you the manuscript PDF, repository link, and the endorsement code that arXiv issued to me.

Best regards,
Ibuki Nagase
Osaka Electro-Communication University
contact@enludus.com
```

```text
現在のプロジェクトには、prototype DSL / runtime、viewer と diagram-to-logic round-trip、semantics の初期整理、underdetermined-world convergence、visibility を用いた geometry 例まで含まれています。

もし先生が該当カテゴリで endorsement 可能で、内容をご覧いただいたうえで問題ないようでしたら、endorsement をお願いできないかと思いご連絡しました。

必要であれば、論文 PDF、GitHub リポジトリ、arXiv から発行された endorsement code をお送りします。

どうぞよろしくお願いいたします。

Ibuki Nagase
Osaka Electro-Communication University
contact@enludus.com
```

## Minimal Package To Send

- `paper/main-public.pdf`
- repository URL
- one-paragraph project summary
- endorsement code from arXiv
