# 2026-03-27: Phase J Visibility Network

This pass extends the visibility pillar from coordination into small role assignment networks.

The core idea is that line of sight is no longer only used to:

- validate a world,
- flip one pursuit/search branch,
- hand off between two target-specific continuations, or
- resolve several deferred entities at once.

It can now also distribute target-specific roles across several agents in the same world.

This matters because it pushes the geometry story closer to a genuine world-model claim:
visibility begins to act like a routing condition over admissible development rather than a local branch guard.
