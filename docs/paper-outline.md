# Paper Outline Draft

## Working Title

World-Oriented Programming:
A Programming Paradigm for Executable World Description Through Space, Time, and Constraints

## Provisional Venue

Onward! at SPLASH

## Alternative Titles

- A Spatial-Temporal Constraint Programming Paradigm
- Programming by Defining Worlds
- Visual-Logical World Description Language
- Continuous-Time Declarative Programming Model

## Draft Abstract

This research proposes a new programming paradigm aligned with human intuitive problem solving.
Conventional programming languages are organized around procedural execution order and explicit state updates, which makes it difficult to directly express spatial relations, continuous time, and logical constraints.

The proposed paradigm, World-Oriented Programming, treats programs as world definitions rather than instruction sequences.
Entities, spatial structure, temporal evolution, and validity conditions are expressed directly, and execution is understood as the development of a constrained world over time.
The model combines first-class spatial constructs, continuous-time progression, global asynchrony with local synchrony, and a constraint-first treatment of exceptions and interactions.

This work aims to provide a foundation for new computational environments in simulation, game design, mathematical modeling, robotics, and educational systems.

## Research Questions

1. Can a world-description language better align with human reasoning than stepwise procedural programming for spatial-temporal tasks?
2. Can continuous time and local synchronization be integrated into a declarative world model without sacrificing coherent observation?
3. Can diagrams, equations, and logical constraints be unified around the same executable world state?

## Core Contributions

1. A conceptual model for world-oriented programming
2. A runtime model based on continuous time, local synchronization, and constrained evolution
3. An initial declarative DSL for describing executable worlds
4. A research path toward visual-logical programming over a shared world model
