# Reference Notes

This file collects citation candidates and short notes about why they matter for the paper.

## Core Related Work

### Jaffar and Lassez, 1987

- Title: `Constraint logic programming`
- Why it matters: strongest conceptual precedent for declarative constraint-first computation
- Source: IBM Research publication page
- Link: https://research.ibm.com/publications/constraint-logic-programming

### Hudak, 1999

- Title: `Functional Reactive Programming`
- Why it matters: central reference for time-varying behavior and event-based reactivity
- Source: Springer chapter page
- Link: https://doi.org/10.1007/3-540-49099-X_1

### Elliott and Hudak, 1997

- Title: `Functional Reactive Animation`
- Why it matters: earlier FRP reference tied to animation and declarative time-varying behavior
- Source note: cited from the Springer FRP chapter reference list
- Link: https://link.springer.com/chapter/10.1007/3-540-49099-X_1

### Modelica Association, 2023

- Title: `Modelica Language Specification version 3.6`
- Why it matters: closest large-scale precedent for declarative model execution without manual variable-solving order
- Source: official specification
- Link: https://specification.modelica.org/maint/3.6/

### Hewitt, Bishop, and Steiger, 1973

- Title: `A Universal Modular ACTOR Formalism for Artificial Intelligence`
- Why it matters: classic asynchronous computation reference for decentralized progression and coordination
- Source note: metadata confirmed via ResearchGate page pointing to IJCAI 1973 / DBLP source
- Link: https://www.researchgate.net/publication/220812785_A_Universal_Modular_ACTOR_Formalism_for_Artificial_Intelligence

### GeoGebra

- Title: `About GeoGebra`
- Why it matters: concrete precedent for visual-mathematical integration
- Source: official GeoGebra about page
- Link: https://stage.geogebra.org/about

### NI LabVIEW

- Title: `LabVIEW Overview`
- Why it matters: practical graphical dataflow comparison point
- Source: official NI documentation
- Link: https://www.ni.com/docs/en-IO/bundle/labview/page/labview-overview.html

## Suggested Citation Roles In The Paper

- Comparison paragraph on non-sequential execution: LabVIEW and FRP
