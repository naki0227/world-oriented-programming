# Decision 0001: Project Identity

## Status

Accepted

## Date

2026-03-24

## Decision

Adopt the following initial names:

- language: `sekai`
- core engine: `orbis`
- paradigm: World-Oriented Programming
- file extension: `.sk`

## Rationale

These names reflect the project's emphasis on world definition, spatial meaning, and temporal evolution while remaining concise and memorable.

## Consequences

- documents should use these names consistently unless a later decision revises them
- examples and future CLI sketches can assume the `sekai` command
